# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-04-10

### Added
- GitHub PR inbox dashboard — single-page browser app for tracking pull requests across multiple repositories, with modes for review-requested, reviewed, own PRs, and other
- Auto-refresh, draft toggle, repo prefix filters, CI status badges, unread PR notifications (favicon + tab title), and per-PR seen tracking via localStorage
- ESLint and Stylelint setup based on `@ringieraxelspringer` shared configs
